#File Organizer
A sleek, modern web-based interface for staging and organizing files. Built entirely with vanilla HTML, CSS, and JavaScript, this project features a beautiful glassmorphism design, animated gradient backgrounds, and intuitive file management controls.

✨ Features
Modern UI/UX: Utilizes glassmorphism (frosted glass effect) and a continuous animated mesh gradient background for a premium feel.

File Staging: Select and stage multiple files using the custom-styled file input.

Quick Clear: A dynamic clear button (×) appears when files are selected, allowing users to quickly reset their current input.

Upload Management: Click "Upload" to move selected files into the organized list.

Remove Items: Delete individual files from the uploaded list seamlessly.

Custom Scrollbars: Clean, stylized scrollbars for navigating long lists of files.

🚀 Future Enhancements (To-Do)

🛠️ Technologies Used
HTML5: Semantic structure.

CSS3: Advanced styling including backdrop-filter for glassmorphism, @keyframes for the animated background, and custom pseudo-elements (::file-selector-button).

Vanilla JavaScript: DOM manipulation, event handling, and array-based state management (selectedFiles).
